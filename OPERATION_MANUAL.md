# 管理画面 操作マニュアル
**対象：サロンオーナー向け**

---

## 1. このマニュアルの使い方

このマニュアルは、LINEのお客さまからの自動返答と、サロンの情報管理をするための「管理画面」の使い方をまとめたものです。

**管理画面へのアクセス**
ブラウザ（Chrome など）で以下のアドレスを開いてください。

```
https://（開発者から案内されたURL）/admin
```

ログイン画面が表示されたら、開発者から渡されたパスワードを入力して「ログイン」を押してください。

[ここに画面キャプチャ：ログイン画面]

ログインに成功するとダッシュボードが表示されます。画面上部の「管理画面」ロゴを押すといつでもダッシュボードに戻れます。

[ここに画面キャプチャ：ダッシュボード画面]

ダッシュボードには以下の3つの数字が表示されます。

| 表示 | 意味 |
|---|---|
| FAQ 件数 | 登録しているよくある質問の総数 |
| 本日の問い合わせ | 今日 LINE で届いたメッセージの件数 |
| エスカレーション件数 | bot が「わからない」と判断して通知を送った累計件数 |

## 2. FAQの追加・編集・削除

FAQは「よくある質問と回答」の一覧です。ここに登録した内容をもとに、bot がお客さまへ返答します。

### 追加する

1. ダッシュボードの「FAQ 管理」を押す
2. 右上の「＋ 新規作成」を押す

[ここに画面キャプチャ：FAQ一覧画面]

3. 以下の3つを入力して「保存する」を押す

| 項目 | 入力内容 | 文字数上限 |
|---|---|---|
| 質問 | お客さまが送ってきそうな言葉 | 500文字 |
| 回答 | bot が返す内容 | 2,000文字 |
| カテゴリ | 「営業時間」「料金」など分類名 | 50文字 |

[ここに画面キャプチャ：FAQ新規作成フォーム]

> **ポイント：** 質問はお客さまが実際に使う言葉で書くと、bot がより正確に答えられるようになります。例）「カットの予約はどうすればいいですか？」

### 編集する

1. FAQ一覧で変更したい行の「編集」を押す
2. 内容を書き直して「更新する」を押す

### 削除する

1. FAQ一覧で削除したい行の「削除」を押す
2. 確認のメッセージが出るので「OK」を押す

> ⚠️ 削除すると元に戻せません。削除してよいか確認してから操作してください。

## 3. お知らせ配信の方法

これまでに LINE でメッセージをやり取りしたことがあるお客さま全員に、一斉にお知らせを送る機能です。

### 配信手順

1. ダッシュボードの「お知らせ配信」を押す
2. テキスト欄にお知らせ内容を入力して「確認画面へ進む」を押す

[ここに画面キャプチャ：メッセージ入力画面]

3. **確認画面**が表示されます。送信対象の人数・内容を必ず確認してください

[ここに画面キャプチャ：送信確認画面]

4. 問題なければ「送信する」を押す
5. 完了画面に「送信成功：○件」が表示されれば完了

> ⚠️ **誤送信に注意：** 「確認画面へ進む」を押しただけでは送信されません。確認画面で内容・人数をよく確かめてから「送信する」を押してください。内容が違う場合は「キャンセル（戻る）」で入力画面に戻れます。

### 別のメッセージを送りたいとき

完了画面の「別のメッセージを送る」を押すと、入力画面に戻ります。

## 4. メニュー・料金の編集

サロンのメニューと料金の一覧を管理できます。ここに登録した内容も、bot の返答に活用されます。

### 追加する

1. ダッシュボードの「メニュー管理」を押す
2. 右上の「＋ 新規作成」を押す
3. 以下を入力して「保存する」を押す

| 項目 | 入力内容 |
|---|---|
| メニュー名 | 例：カット、カラー、パーマ |
| 料金（円） | 半角数字のみ。0以上の整数で入力 |
| 説明 | 施術内容や所要時間など |

[ここに画面キャプチャ：メニュー新規作成フォーム]

> **料金の入力ルール：** 「3500」のように数字だけ入力してください。「¥」や「円」は不要です。小数（例：3500.5）や、マイナス（例：-100）は入力できません。

### 編集する

1. メニュー一覧で変更したいメニューの「編集」を押す
2. 内容を書き直して「更新する」を押す

### 削除する

1. メニュー一覧で削除したいメニューの「削除」を押す
2. 確認のメッセージが出るので「OK」を押す

## 5. 会話ログ・エスカレーション通知の確認

### 会話ログを見る

1. ダッシュボードの「会話ログ」を押す
2. お客さまごとに最後のメッセージと日時が表示されます

[ここに画面キャプチャ：会話ログ一覧]

> **赤いバッジ「エスカレーション」** が付いているお客さまは、bot が「うまく答えられなかった」と判断して、オーナーに通知を送った履歴があります。

3. 名前（ユーザーID）の行を押すと、そのお客さまとの会話の全履歴が確認できます

[ここに画面キャプチャ：会話詳細画面]

> **オレンジのバッジ「エスカレーション済み」** が付いている行が、bot が答えられなかったメッセージです。

### エスカレーション通知とは

お客さまからのメッセージに bot が自信を持って答えられない場合、オーナーの LINE に以下のような通知が届きます。

```
【エスカレーション通知】

■ 質問内容
（お客さまのメッセージ）

■ 確信度
低（スコア：3/10）

■ AI回答案
（bot が考えた回答の案）
```

通知を受け取ったら、お客さまに直接 LINE で返信してください。

## 6. botが変な返答をしたときの対処

bot の返答が不正確だったり、엉뚱한 内容だったりする場合、ほとんどの原因はFAQの情報不足です。以下の手順で改善できます。

**手順 1：会話ログを確認する**
「会話ログ」を開き、どんな質問に対して問題が起きたかを確認します。

**手順 2：FAQを追加・修正する**
「FAQ管理」を開き、その質問に対応するFAQを追加または修正します。
- 似たような質問が複数あれば、まとめて1つのFAQにするより、それぞれ別のFAQに分けた方が正確になります。

**手順 3：様子を見る**
FAQ保存後、同じ内容の質問を LINE から送って返答を確認してください。改善されていれば完了です。

> それでも改善しない場合や、bot が全く返答しなくなった場合は、下記の問い合わせ先にご連絡ください。
